Subgrid für komplexe Layouts

Mit CSS Subgrid können wir verschachtelte Grids erstellen, die die Raster-Linien des übergeordneten Grids erben. Dies ermöglicht konsistente Ausrichtungen über mehrere Ebenen hinweg.

Hinweis: CSS Subgrid wird von modernen Browsern unterstützt. Falls die Beispiele nicht wie erwartet angezeigt werden, stellen Sie sicher, dass Sie einen aktuellen Browser verwenden.

Was ist Subgrid?

Subgrid ist eine Erweiterung des CSS Grid-Layouts, die es ermöglicht, dass ein Grid-Element, das selbst ein Grid ist, die Rasterlinien des übergeordneten Grids erbt. Dies ist besonders nützlich, wenn Sie komplexe, verschachtelte Layouts erstellen und dabei eine konsistente Ausrichtung beibehalten möchten.

.parent-grid { display: grid; grid-template-columns: repeat(3, 1fr); gap: 20px; } .child-element { grid-column: span 2; /* Subgrid definieren */ display: grid; grid-template-columns: subgrid; /* Alternative für Zeilen */ grid-template-rows: subgrid; }

Beispiel 1: Produktkarten mit konsistenter Höhe

In diesem Beispiel verwenden wir Subgrid, damit alle Produktkarten die gleiche Höhe haben und ihre inneren Elemente (Bild, Info, Aktionen) horizontal ausgerichtet sind, auch wenn der Inhalt unterschiedlich lang ist.

.product-grid { display: grid; grid-template-columns: repeat(3, 1fr); gap: 20px; } .product-card { display: grid; grid-template-rows: auto 1fr auto; /* Subgrid für Zeilen */ grid-row: span 3; grid-template-rows: subgrid; }

Beispiel 2: Formulare mit ausgerichteten Labels

Mit Subgrid können wir Formulargruppen erstellen, bei denen die Labels und Eingabefelder über verschiedene Abschnitte hinweg ausgerichtet sind.

.form-grid { display: grid; grid-template-columns: 150px 1fr; gap: 10px 20px; } .form-section { grid-column: 1 / -1; /* Subgrid für Spalten */ display: grid; grid-template-columns: subgrid; grid-column: span 2; }

.page-layout { display: grid; grid-template-columns: 250px 1fr 200px; grid-template-rows: auto 1fr auto; gap: 20px; } .page-sidebar { grid-row: 2 / 3; display: grid; grid-template-rows: subgrid; grid-row: span 1; }

Wann sollte man Subgrid verwenden?

  • Konsistente Ausrichtung: Wenn Elemente in verschiedenen Containern horizontal oder vertikal ausgerichtet werden sollen.
  • Komplexe Formulare: Um Labels und Eingabefelder über mehrere Abschnitte hinweg auszurichten.
  • Responsive Cards: Damit Karten-Elemente in verschiedenen Größen die gleichen Proportionen beibehalten.
  • Multi-Level-Navigation: Für komplexe Navigationsstrukturen mit einheitlicher Ausrichtung.

Browser-Unterstützung

CSS Subgrid wird von den aktuellen Versionen aller modernen Browser unterstützt:

  • Chrome: Ab Version 117 (September 2023)
  • Firefox: Ab Version 71 (Dezember 2019)
  • Safari: Ab Version 16 (September 2022)
  • Edge: Ab Version 117 (September 2023)

Für ältere Browser kann als Fallback ein normales Grid ohne Subgrid-Funktionalität verwendet werden.

CSS Subgrid ist Teil der CSS Grid Layout Module Level 2 Spezifikation und ermöglicht noch leistungsfähigere und flexiblere Layouts.
