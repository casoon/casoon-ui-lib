@property Definitionen

Mit @property können benutzerdefinierte CSS-Eigenschaften deklariert werden, die sich animieren lassen:

Hover mich für animierte Eigenschaften

Dieses Element verwendet @property-Definitionen für animierbare benutzerdefinierte Eigenschaften wie Farbe, Rotation und Skalierung.

Hover mich für animierte Gradienten

Dieses Element verwendet @property-Definitionen für animierbare Gradienten mit Übergängen zwischen verschiedenen Farben und Winkeln.

Mit @property können Übergänge zwischen Werten realisiert werden, die mit herkömmlichen CSS-Übergängen nicht möglich wären, wie z.B. Farben, Winkel, Zahlen und Längen mit unterschiedlichen Einheiten.

Code-Beispiel:

/* @property-Definitionen */
@property --color-primary {
  syntax: '<color>';
  inherits: false;
  initial-value: var(--color-primary, #3b82f6);
}

@property --radius {
  syntax: '<length-percentage>';
  inherits: false;
  initial-value: var(--border-radius-md, 0.5rem);
}

@property --rotation-angle {
  syntax: '<angle>';
  inherits: false;
  initial-value: 0deg;
}

/* Verwendung in CSS */
.animated-card {
  background-color: var(--color-primary);
  border-radius: var(--radius);
  transform: rotate(var(--rotation-angle)) scale(var(--scale-factor));
  transition: 
    --color-primary 0.3s ease, 
    --radius 0.3s ease, 
    --rotation-angle 0.5s ease;
}

.animated-card:hover {
  --color-primary: var(--color-primary-dark, #4f46e5);
  --radius: var(--border-radius-lg, 1rem);
  --rotation-angle: 5deg;
}
