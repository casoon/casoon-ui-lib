Relationale Selektoren (:has) - Casoon UI

Beispiele für die Verwendung von CSS :has() für kontextbezogene Styling-Änderungen

Formulare mit :has()

.form-group:has(:invalid) { border-color: var(--color-error); } .form-group:has(:focus) { border-color: var(--color-primary); }
Bitte geben Sie einen gültigen Namen ein
Die Nachricht ist gültig

Fallback für Browser ohne :has()-Unterstützung: Utility-Klassen wie .has-invalid, .has-valid

Karten mit :has()

.card:has(.featured) { border-color: var(--color-accent); } .card:has(a) { cursor: pointer; } .card:has(.image) { padding-top: 0; }

Standard Karte

Diese Karte hat keine besonderen Eigenschaften.

Featured

Featured Karte

Diese Karte hat einen featured-Marker und erhält automatisch einen akzentuierten Rahmen.

Karte mit Bild

Diese Karte enthält ein Bild und hat keinen oberen Innenabstand.

Interaktive Karte

Diese Karte enthält einen Button und wird dadurch automatisch anklickbar.

Listen und Navigation mit :has()

ul:has(> li:hover) { background-color: var(--color-gray-50); } .nav-item:has(.active) { font-weight: bold; }
  • Startseite
  • Produkte
  • Über uns
  • Kontakt
  • Hover über diesen Eintrag, um die gesamte Liste hervorzuheben
  • Die Liste ändert ihren Hintergrund, wenn ein Element gehovered wird
  • Drittes Listenelement

Bedingte Stile ohne JavaScript

.parent:has(.child:checked) .conditional { display: block; } .accordion:has(input:checked) .content { max-height: 500px; }
Dieser Inhalt wird angezeigt, wenn die Checkbox aktiviert ist. Ohne JavaScript, nur mit CSS :has()!
