Casoon UI Library - Beispiele

Willkommen zur Casoon UI Library! Diese Bibliothek bietet eine Sammlung von wiederverwendbaren Komponenten, Animationen und Effekten für moderne Webanwendungen.

Animationen

Entdecken Sie unsere Sammlung von Animationen, von einfachen Fade-Effekten bis hin zu komplexen Bewegungsabläufen. Alle Animationen sind mit OpenProps erstellt und vollständig anpassbar.

Animationen ansehen

Visuelle Effekte

Beeindruckende visuelle Effekte wie Neon-Glühen, Glitch-Effekte und Gradient-Text. Perfekt für auffällige Überschriften und besondere Design-Elemente.

Effekte ansehen

Grid-System

Unser flexibles Grid-System basiert auf OpenProps und bietet responsive Layout-Möglichkeiten für jede Bildschirmgröße. Mit zahlreichen Utility-Klassen für die präzise Kontrolle des Layouts.

Grid-System ansehen

Typografie

Eine umfassende Typografie-System mit responsiven Schriftgrößen, verschiedenen Schriftfamilien und speziellen Textformaten wie Pull-Quotes und Initialen.

Typografie ansehen

Verwendung

Um die Casoon UI Library in Ihrem Projekt zu verwenden, installieren Sie sie über npm: 

npm install casoon-ui-lib

Importieren Sie dann die gewünschten Styles in Ihrem Projekt: 

import 'casoon-ui-lib/reset.css';
import 'casoon-ui-lib/base.css';
import 'casoon-ui-lib/animations.css';
import 'casoon-ui-lib/effects.css';
