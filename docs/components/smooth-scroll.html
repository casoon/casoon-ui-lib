Optionen

Option Beschreibung Standardwert
duration Dauer der Scroll-Animation in Millisekunden 500
easing Easing-Funktion für die Animation easeInOutQuad
offset Zusätzlicher Offset vom Ziel in Pixeln 0 
// JavaScript Initialisierung
document.addEventListener('DOMContentLoaded', () => {
    const smoothScroll = new SmoothScroll({
        duration: 500,
        easing: 'easeInOutQuad',
        offset: 0
    });
});
