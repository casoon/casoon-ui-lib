Benutzerdefinierte Animationen

Benutzerdefinierte Animation

/* CSS für benutzerdefinierte Animation */
@keyframes custom-animation {
    0% {
        transform: scale(1);
        opacity: 1;
    }
    50% {
        transform: scale(1.2);
        opacity: 0.5;
    }
    100% {
        transform: scale(1);
        opacity: 1;
    }
}

.animation-custom {
    animation: custom-animation 2s infinite;
}
