Übersicht

Die Casoon UI Library basiert auf einem modularen System von Basis-Komponenten und Design-Tokens. Diese Grundlagen bilden das Fundament für alle weiteren Komponenten und Styles.

Design Tokens

Design Tokens sind die grundlegenden Bausteine unseres Design-Systems. Sie definieren konsistente Werte für:

  • Farben
  • Schriftarten und Typografie
  • Abstände und Größen
  • Schatten und Effekte
  • Übergänge und Animationen
/* Beispiel für Design Tokens */
:root {
    --color-primary: #007bff;
    --color-background: #ffffff;
    --font-family-base: system-ui, -apple-system, sans-serif;
    --space-4: 1rem;
    --shadow-md: 0 4px 6px rgba(0, 0, 0, 0.1);
}

Basis-Module

Reset

Ein moderner CSS Reset, der Browser-spezifische Stile zurücksetzt und eine konsistente Basis schafft.

Typografie

Definiert Schriftarten, Größen, Gewichte und Zeilenabstände für eine konsistente Textdarstellung.

Layout

Bietet ein flexibles Grid-System und grundlegende Layout-Komponenten für die Seitenstruktur.

Utilities

Hilfreiche Utility-Klassen für schnelles Styling und Layout-Anpassungen.

Verwendung

Die Basis-Module können einzeln oder als Bundle importiert werden:

/* Einzelne Module importieren */
@import "core/tokens.css";
@import "core/reset.css";
@import "core/typography.css";
@import "core/layout.css";
@import "core/utilities.css";

/* Oder das komplette Bundle */
@import "core.css";

Best Practices

  • Verwenden Sie die Design Tokens für konsistente Werte
  • Nutzen Sie die Utility-Klassen für schnelle Anpassungen
  • Erweitern Sie das System durch neue Tokens und Utilities
  • Halten Sie sich an die vordefinierten Breakpoints für responsive Designs
