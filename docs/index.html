Willkommen

Dies ist die Dokumentation des Casoon UI Designsystems. Hier finden Sie alle Informationen zu den Komponenten, Mustern und Grundlagen des Systems.

Schnellstart

Installation

Installieren Sie das Designsystem über npm oder yarn.

npm install @casoon/ui

Verwendung

Importieren Sie die Styles in Ihr Projekt.

@import '@casoon/ui/core/components.css';

Beispiel

Verwenden Sie die Komponenten in Ihrem HTML.

<button class="button">Klick mich</button>
