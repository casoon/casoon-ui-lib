JavaScript-Integration

Beispiel für die manuelle Initialisierung und Konfiguration:

// Smooth Scroll initialisieren
const smoothScroll = new SmoothScroll({
    // Scroll-Geschwindigkeit in Millisekunden
    speed: 800,
    // Offset für fixierte Header
    offset: 100,
    // Easing-Funktion
    easing: 'easeInOutCubic'
});

// Zu einem Element scrollen
smoothScroll.scrollTo('#target');

// Event-Listener für Scroll-Events
document.addEventListener('scrollStart', () => {
    console.log('Scroll gestartet');
});

document.addEventListener('scrollEnd', () => {
    console.log('Scroll beendet');
});

Browser-Kompatibilität

Der Smooth-Scroll-Effekt wird von allen modernen Browsern unterstützt. Für ältere Browser wird ein Fallback auf das Standard-Scrollverhalten verwendet.

  • Chrome: ✓ Unterstützt
  • Firefox: ✓ Unterstützt
  • Safari: ✓ Unterstützt
  • Edge: ✓ Unterstützt
  • IE11: Fallback
