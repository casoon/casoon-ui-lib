Benutzerdefinierte Animationen

Beispiel für die Erstellung einer benutzerdefinierten Animation:

@keyframes custom-animation {
    0% {
        transform: scale(1);
        opacity: 1;
    }
    50% {
        transform: scale(1.2);
        opacity: 0.5;
    }
    100% {
        transform: scale(1);
        opacity: 1;
    }
}

.custom-animation {
    animation: custom-animation 2s infinite;
}

JavaScript-Integration

Beispiel für die dynamische Anwendung von Animationen:

const element = document.querySelector('.element');

// Animation hinzufügen
element.classList.add('fade-in');

// Animation nach Abschluss entfernen
element.addEventListener('animationend', () => {
    element.classList.remove('fade-in');
});

// Animation mit Verzögerung starten
setTimeout(() => {
    element.classList.add('slide-in-right');
}, 1000);

Alpine.js Integration

Beispiel für die Verwendung mit Alpine.js:

<div 
    x-data="{ isAnimated: false }"
    @click="isAnimated = !isAnimated"
    :class="{ 'fade-in': isAnimated }">
    Klicken Sie für Animation
</div>
