Übersicht

Die Casoon UI Library bietet eine Vielzahl von vorgefertigten Komponenten für moderne Webanwendungen. Diese Komponenten bauen auf den Grundlagen auf und können einfach in Ihre Projekte integriert werden.

Basis-Komponenten

Buttons

Verschiedene Button-Varianten mit unterschiedlichen Stilen und Zuständen.

Cards

Flexible Karten-Komponenten für die Darstellung von Inhalten.

Badges

Status- und Label-Badges für verschiedene Anwendungsfälle.

Navigations-Komponenten

Sidebar

Eine flexible Navigations-Sidebar mit verschiedenen Layout-Optionen.

Hamburger Menu

Animierte Hamburger-Menü-Icons für mobile Navigation.

Formular-Komponenten

Formulare

Vollständige Formular-Komponenten mit verschiedenen Eingabetypen.

Slider

Range-Slider für numerische Eingaben und Einstellungen.

Effekte und Animationen

Effekte

Spezialeffekte und Hover-States für interaktive Elemente.

Animationen

Verschiedene Animationen für Übergänge und Interaktionen.

Smooth Scroll

Sanfte Scroll-Effekte für eine bessere Benutzererfahrung.

Verwendung

Die Komponenten können einzeln oder als Bundle importiert werden:

/* Einzelne Komponenten importieren */
@import "components/button.css";
@import "components/card.css";
@import "components/badge.css";
@import "components/sidebar.css";
@import "components/slider.css";
@import "components/animations.css";

/* Oder das komplette Bundle */
@import "core.css";

Best Practices

  • Verwenden Sie die Komponenten konsistent in Ihrem Projekt
  • Passen Sie die Komponenten mit den bereitgestellten Variablen an
  • Kombinieren Sie Komponenten für komplexere UI-Elemente
  • Testen Sie die Komponenten auf verschiedenen Geräten und Browsern
