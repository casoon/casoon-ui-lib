Grid-Utility-Klassen

Kontrollieren Sie das Grid-Layout mit diesen Utility-Klassen.

Spalten

  • cols-1 bis cols-12 - Spaltenanzahl
  • sm:cols-* - Spalten für kleine Bildschirme
  • md:cols-* - Spalten für mittlere Bildschirme
  • lg:cols-* - Spalten für große Bildschirme

Spannen

  • col-span-1 bis col-span-12 - Spalten spannen
  • sm:col-span-* - Spannen für kleine Bildschirme
  • md:col-span-* - Spannen für mittlere Bildschirme
  • lg:col-span-* - Spannen für große Bildschirme

Ausrichtung

  • items-start - Oben ausrichten
  • items-center - Mittig ausrichten
  • items-end - Unten ausrichten
  • justify-start - Links ausrichten
  • justify-center - Zentrieren
  • justify-end - Rechts ausrichten

Abstände

  • gap-1 bis gap-8 - Spaltenabstand
  • gap-x-* - Horizontaler Abstand
  • gap-y-* - Vertikaler Abstand
