Effekt-Utility-Klassen

Kontrollieren Sie die Effekte mit diesen Utility-Klassen.

Intensität

  • effect-strong - Starker Effekt
  • effect-medium - Mittlere Intensität
  • effect-soft - Sanfter Effekt

Animation

  • effect-animated - Animierter Effekt
  • effect-pulse - Pulsierender Effekt
  • effect-hover - Hover-Effekt
