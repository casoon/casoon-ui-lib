Animations-Utility-Klassen

Kontrollieren Sie die Animationen mit diesen Utility-Klassen.

Geschwindigkeit

  • animate-slow - Langsame Animation
  • animate-normal - Normale Geschwindigkeit
  • animate-fast - Schnelle Animation

Wiederholung

  • animate-once - Einmalige Ausführung
  • animate-infinite - Endlose Wiederholung
  • animate-repeat-2 - Zweimalige Ausführung
  • animate-repeat-3 - Dreimalige Ausführung

Verzögerung

  • animate-delay-100 - 100ms Verzögerung
  • animate-delay-200 - 200ms Verzögerung
  • animate-delay-300 - 300ms Verzögerung

Timing-Funktion

  • animate-ease - Standard Timing
  • animate-ease-in - Langsamer Start
  • animate-ease-out - Langsames Ende
  • animate-ease-in-out - Langsamer Start und Ende
