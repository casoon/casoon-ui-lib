Casoon UI Library

Eine moderne, flexible UI-Bibliothek mit Fokus auf Animationen, Effekte und responsive Layouts.

Über diese Bibliothek

Die Casoon UI Library ist eine vielseitige UI-Bibliothek, die sowohl in Astro JS Projekten als auch in anderen Webprojekten eingesetzt werden kann:

Astro JS Integration

Optimiert für Astro JS mit speziellen Komponenten wie FontLoader und Layout.

Beispiel: import { FontLoader } from 'casoon-ui-lib/components';

CSS-Only Verwendung

Kann auch ohne Astro JS verwendet werden, nur mit CSS-Klassen.

Beispiel: <link rel="stylesheet" href="path/to/casoon-ui-lib/dist/styles.css">
